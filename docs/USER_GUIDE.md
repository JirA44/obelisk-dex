## Connexion du Wallet

### Wallets Supportés
- MetaMask
- WalletConnect
- Coinbase Wallet
- Rabby
- Trust Wallet

### Comment se Connecter

1. Cliquez sur **"Connect Wallet"** en haut à droite
2. Sélectionnez votre wallet
3. Approuvez la connexion dans votre wallet
4. Signez le message d'authentification (gratuit, pas de gas)

> **Important**: OBELISK est **non-custodial**. Nous n'avons jamais accès à vos clés privées.

## Trading

### Passer un Ordre

1. Sélectionnez la paire (ex: BTC/USDC)
2. Choisissez le type d'ordre:
   - **Market**: Exécution immédiate au prix du marché
   - **Limit**: Exécution au prix que vous définissez
3. Entrez le montant
4. Cliquez sur **Buy** ou **Sell**
5. Confirmez dans votre wallet

### Order Book

L'order book affiche:
- **Vert (Bids)**: Ordres d'achat
- **Rouge (Asks)**: Ordres de vente
- **Spread**: Différence entre meilleur achat et meilleure vente

### Frais de Trading

| Type | Frais |
|------|-------|
| Maker | 0.05% |
| Taker | 0.10% |

## Outils Avancés

### Grid Trading Bot

Trade automatiquement dans une range de prix:
1. Définissez prix min/max
2. Nombre de grilles (5-50)
3. Montant total à investir
4. Le bot achète bas et vend haut automatiquement

### Copy Trading

Copiez les meilleurs traders:
1. Consultez le leaderboard
2. Sélectionnez un trader
3. Définissez le % de votre capital
4. Chaque trade est copié automatiquement

### Trailing Stop

Protégez vos gains:
1. Définissez le % de trailing (ex: 5%)
2. Si le prix monte, le stop monte
3. Si le prix baisse de 5%, vente automatique

### Arbitrage Scanner

Trouvez les opportunités d'arbitrage entre DEX:
- Uniswap vs SushiSwap
- Cross-chain (Arbitrum ↔ Optimism)
- CEX vs DEX

## FAQ

### Général

**Q: OBELISK est-il sûr?**
R: Oui. Nous sommes non-custodiaux, vos fonds restent dans votre wallet.

**Q: Quels réseaux sont supportés?**
R: Arbitrum (principal), Optimism, Ethereum mainnet, Polygon.

**Q: Y a-t-il un token OBELISK?**
R: Pas encore. Suivez nos annonces.

### Trading

**Q: Pourquoi mon ordre n'est pas exécuté?**
R: Pour les ordres limit, le prix doit atteindre votre niveau. Vérifiez l'order book.

**Q: Comment annuler un ordre?**
R: Allez dans "Open Orders" et cliquez sur "Cancel".

**Q: Les frais sont-ils en plus du montant?**
R: Les frais sont déduits du montant reçu.

### Investissements

**Q: Puis-je perdre mon capital avec les Vaults?**
R: Non, les Protected Vaults garantissent votre capital (90-100% selon le vault).

**Q: Comment sont générés les rendements?**
R: Staking, lending, et stratégies DeFi diversifiées.

**Q: Y a-t-il des périodes de lock?**
R: Le Liquid Staking est instantané. Les Bonds ont une maturité fixe.

### Technique

**Q: Pourquoi je vois "Connecting..."?**
R: Vérifiez votre connexion internet. Rafraîchissez la page.

**Q: Mon wallet ne se connecte pas**
R:
1. Vérifiez que vous êtes sur le bon réseau (Arbitrum)
2. Essayez de déconnecter/reconnecter
3. Videz le cache du navigateur

**Q: Les prix ne se mettent pas à jour**
R: Cliquez sur le bouton de refresh ou F5. Si le problème persiste, vérifiez l'indicateur de connexion.
