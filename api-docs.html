POST /api/auth/session Create authenticated session

Requires X-API-Key header with your API key.

Body ParameterTypeDescription
ttlMinutes integer Session lifetime (default: 60, max: 1440)

üìà Trading

POST /api/order Place market or limit order

Requires authentication (API Key + Session).

Body ParameterTypeDescription
pair required string Trading pair (e.g., BTC/USDC)
side required string buy | sell
quantity required number Amount to trade
type string market (default) | limit
leverage number 1-100 (default: 1)
GET /api/positions Get all open positions

Requires authentication. Returns all open positions with P&L.
